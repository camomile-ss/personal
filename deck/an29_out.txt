IEEE802.1X認証で、PCをリピータハブ経由で認証を行うスイッチに接続してしまうとセキュリティ上の問題が残る。その問題は何か。	認証されたポート経由で認証されていないPCの接続が許可されてしまう。<br>正規ユーザが認証に成功すると、スイッチのそのポートがオープンになるので。	network
検疫ネットワーク	セキュリティに問題があるPCを社内ネットワークなどに接続させないことを目的とした仕組み。<br>ウィルスに感染してないか確認したり、パターンファイルやパッチ更新情報を自動的に検査して強制的な更新を行ったりする。<br>認証スイッチ方式やDHCPサーバ方式などがある。	network
FWとIDS/IPSの違い	FWはヘッダ情報のみ確認。IDS/IPSはデータも確認。	network
FWでは防げない攻撃の例	クロスサイトスクリプティング, バッファオーバフロー	network
IDSの種類	<ul><li>NIDS: Network-based IDS<br>ネットワーク上の通信を監視</li><li>HIDS: Host-based IDS<br>コンピュータ上の以上を監視</li></ul>	network
IDS/IPSの不正侵入の検知方法	<ul><li>シグネチャ型: 既知の攻撃パターンに基づく攻撃とマッチングすることにより検知</li><li>アノマリ型: RFCのプロトコル仕様などと比較して異常なパケットやトラフィックを分析して統計的に異常なパケットを攻撃として検知。</li></ul>	network
IDS/IPSの誤検知と見逃し	<ol><li>フォールスポジティブ（誤検知）: 正常な通信を誤って異常と検知</li><li>フォールスネガティブ（見逃し）: 異常な通信を検知できず見逃し</li></ol>	network
